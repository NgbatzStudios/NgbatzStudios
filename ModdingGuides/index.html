Introduction

Welcome to the modding guide! Here you'll learn how to mod your game step by step.

Setup

To begin modding, ensure you have the core mods installed and NG Launcher (optional) ready for the next steps.

Modding Basics

Learn how to install core mods and other peoples mods!

You can find mods either here: https://mod.io/g/escape-poland or here: https://discord.gg/VpvhPTPr4B

For core mods you can find them in the NG Launcher or in the discord which is stated above.

Making Mods

To make mods make sure you have either Visual Studio or Visual Studio Code installed.

Next go to your games files and you will see a folder name "Mods" now go in there.

Once you are in the "Mods" folder create a new folder this will be the name of the mod and where all the mods files will go.

Once you have creaated your mods main folder you can create a new c# script. Once created you will need to open it in Visual Studio or Visual Studio Code

Once opened there will be nothing in the file so now copy and paste this base script which you can find here

Now you see where there is the Namespace change the bit after that to the main folders name and for any scripts in sub-folders do the same but add a dot with the sub-folders name eg. YourFirstMod.FunnyFolder

Once that's done get to coding!

Troubleshooting

Encountering issues? Find solutions to common problems here.

If you are experiencing problems with mods compiling that usually means mods are broken. We try not to do anything that causes modding to break.
